Thư mục này dùng để chứa 3 ảnh minh chứng theo yêu cầu bài tập:

1. pipeline-success.png  -> chụp tại Build > Pipelines
2. pipeline-jobs.png     -> chụp tại Build > Jobs
3. pages-url.png         -> chụp tại Deploy > Pages

Mẫu thao tác:
- Push code lên nhánh main
- Đợi pipeline passed
- Mở các trang trên GitLab và chụp ảnh lưu vào thư mục này
